# Module 09 — Cosmic Data  
Discovering Pydantic Models & Data Validation

## Objective

This module focuses on learning how to validate and structure data using **Pydantic**, one of Python’s most powerful libraries for data validation.

The project simulates a cosmic data observatory where different space systems send structured information that must be validated, cleaned, and processed before being used.

---

## What You Learn

- Creating structured data models using `BaseModel`
- Adding validation rules with `Field`
- Implementing custom validation logic
- Using enums for controlled value sets
- Handling optional fields and default values
- Managing nested models and complex relationships

---

## Learning Outcome

By the end of this module, you should be able to:

- Design robust and validated data models
- Enforce strict data rules using Pydantic
- Implement custom validation logic for complex cases
- Work with nested data structures and relationships
- Build reliable data validation systems

This module develops the essential skills required for **building safe APIs, validating datasets, and designing reliable data pipelines**.
